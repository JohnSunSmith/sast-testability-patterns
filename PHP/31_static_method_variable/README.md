# Static Method Variable

Tags: sast, php, php_v7.4.9

## Description

In PHP a static method can be stored in a variable and than be called from that variable.

This instance shows the use of a static function.

### Code

```PHP
<?php
class myclass {
    static function F($b) {
        return $b;
    }
}

$a = $_GET["p1"]; // source
$func = "F";
$b = myclass::$func($a);
echo $b; // sink
```
